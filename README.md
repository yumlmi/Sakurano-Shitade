# ノベルゲーム「桜の下で」

大学の授業で制作した Unity 製の恋愛ノベルゲームです（4人チーム）。

## 概要
ストーリー中の選択肢によってエンディングが分岐する、恋愛ノベルゲームです。  
AI を活用してストーリーやキャラクターのセリフ、キャラクターの立ち絵・背景画像を制作しました。

これらの取り組みが評価され、  
- オープンキャンパスにて **2度展示**
- 近畿大学情報学部の **公式 X にて授業内制作物の一例として紹介**

していただきました。

## 担当
- ゲームロジック実装  
- UI 制御  
- シーン遷移の実装  
- ストーリー制作  
- 背景・キャラクター立ち絵の選定  

## 技術
- Unity  
- C#
