# ノベルゲーム「桜の下で」

大学の授業で制作した Unity 製の恋愛ノベルゲームです（4人チーム）。

## 概要
ストーリー中の選択肢によってエンディングが分岐する、恋愛ノベルゲームです。
AI を活用してストーリーとキャラクターのセリフ、キャラクターの立ち絵や背景画像などを作ったことを高く評価していただき、オープンキャンパスで２度展示していただきました。
また、近畿大学情報学部の公式 X にて授業内の制作物の一例として紹介していただきました。

## 担当
- ゲームロジック実装  
- UI 制御  
- シーン遷移の実装  
- ストーリー制作  
- 背景やキャラクター立ち絵の選定  

## 技術
- Unity  
- C#
